# Dima Launcher

This is a C++ application to launch Dima main programme.
When you type `$ dima`, the application will launch `$ dinfio <dima_path>/main.fio`

## Compilation

- Linux x86_64:
```
$ ./compile_linux.sh
```

- Linux ARMv7:
```
$ ./compile_linux_armv7.sh
```

- Linux Docker:
```
$ ./compile_linux_docker.sh
```

- macOS:
```
$ ./compile_macos.sh
```

- Windows:
```
> ./compile_windows.bat
```
